Student Utility CLI

This is a simple command-line based Python project built to practice basic programming concepts.

Features
- CGPA Calculator  
- Simple Interest Calculator  
- Compound Interest Calculator  
- Area Calculator (Circle & Triangle)  

Tech Used
- Python  

Purpose
I built this project to strengthen my fundamentals in Python, including functions, loops, and user input handling.

How to Run
1. Download the file  
2. Open terminal in the folder  
3. Run:
   python3 student_utility.py
